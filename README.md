# Andrej Jasso - Personal Portfolio

A modern, responsive portfolio website showcasing my iOS applications and development work.

## About

This is my personal portfolio website that highlights the various mobile applications I've developed. The site features a clean, modern design with smooth animations and responsive layouts that work across all devices.

## Local Development

Simply open `index.html` in your web browser. No build process or dependencies required!

## Deployment

This site can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

### GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to repository Settings
3. Navigate to Pages section
4. Select the branch (usually `main` or `master`)
5. Set the folder to `/` (root)
6. Click Save

Your site will be available at `https://yourusername.github.io/repository-name/`

## Customization

Feel free to customize:
- Update app descriptions in `index.html`
- Modify colors in the `:root` CSS variables in `styles.css`
- Add your own contact information
- Include additional sections or apps
- Add links to App Store listings when apps are published
